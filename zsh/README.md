# dsh, also as dot-zsh

Personal customized zsh config.

## Usage

```
# switch profiles (zsh config)
dsh use [omz|ant|bare|play|dummy]
```

## How it works?

- `task setup`
- new session auto-load `~/.zshrc` as soft-link
