# OrbStack

Replacement of docker-desktop app

- https://docs.orbstack.dev/quick-start

## orb and orbctl

symlink to same file: `scli`, but behavior differently

- `orb` just short command of `orbctl`
- `orbctl` is the final boss

## What's the default machine?

```
orb default
```

## Use which arch?

- most server use x86 arch, so first use `amd64`
- also note Apple M1/2/3 use `arm64` arch, use `arch` to check.
